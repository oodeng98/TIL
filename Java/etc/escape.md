# Escape 문자

### 대표적인 escape 문자

| escape 문자 | 설명            |
| ----------- | --------------- |
| \'          | 작은따옴표 출력 |
| \"          | 큰따옴표 출력   |
| \ \         | 백슬래시 출력   |
| \?          | 물음표 출력     |
| \n          | 줄 바꿈         |
| \t          | 가로 탭         |
| \v          | 세로 탭         |
| \b          | 백스페이스      |
| \r          | 캐리지리턴      |

- escape 문자를 무시하고 싶다면 문자열 앞에 r을 입력
- Ex)

```python
print(r"\"hello world\"")
```

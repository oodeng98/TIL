# 자바 기초 문법

### 출력
- print
  - 줄 넘김 기능 없음
- println
  - 출력 + 한줄(ln -> line)
- printf
  - %d: 10진수 정수(**d**ecimal)
  - %f: 실수(**f**loat)
  - %c: 문자(**c**har)
  - %s: 문자열(**s**tring)
  - %o: 8진수(**o**ctal)
  - %x: 16진수(He**x**adecimal)
    - 10~15를 a~f로 출력
      - %X: A~F로 출력
  - %n?: n칸을 확보한 상태로 우측에 붙여서 출력
- [escape 문자](./escape.md)
